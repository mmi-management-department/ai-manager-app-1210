# 画像アセットフォルダ

このフォルダには、アプリで使用するロゴとアバター画像を配置します。

---

## 📁 配置するファイル

### 1. 会社ロゴ
**ファイル名:** `mm_logo.png`

**推奨仕様:**
- **サイズ:** 幅200-400px、高さ自動
- **形式:** PNG（透過背景推奨）
- **解像度:** 72-150 DPI

**取得方法:**
1. 会社の公式ロゴデータを使用
2. 広報部門から入手
3. または会社Webサイトからダウンロード

---

### 2. AI管理部長アバター
**ファイル名:** `ai_manager_avatar.png`

**推奨仕様:**
- **サイズ:** 200x200px以上（正方形または肖像画サイズ）
- **形式:** PNG（透過背景推奨）
- **スタイル:** プロフェッショナルな女性管理職の写真またはイラスト

**作成方法:**

#### オプション1: AI画像生成（推奨）
**ツール:**
- Stable Diffusion（無料）
- Midjourney（有料）
- DALL-E 3（有料）

**プロンプト例:**
```
cute cartoon business manager avatar, 40s Japanese male, 
wearing glasses, navy suit, friendly smile, 
minimalist flat design, white background, 
professional but approachable, 
PNG transparent background
```

**日本語プロンプト例:**
```
かわいいビジネスマンのアバター、40代の日本人男性、
メガネ、紺色のスーツ、親しみやすい笑顔、
シンプルなフラットデザイン、白背景、
プロフェッショナルだけど親しみやすい、
透過PNG
```

#### オプション2: オンライン無料ツール
**ツール:**
- **Canva** (https://www.canva.com/)
  - テンプレート「Avatar」を選択
  - カスタマイズして保存

- **Adobe Express** (https://www.adobe.com/express/)
  - 無料アバター作成機能

- **Bitmoji** (https://www.bitmoji.com/)
  - カスタムアバター作成

#### オプション3: 写真からイラスト化
**ツール:**
- **Cartoon Photo** (https://cartoon.pho.to/)
- **ToonMe** (https://toonme.com/)
- **Prisma** (https://prisma-ai.com/)

**手順:**
1. 管理部長の写真を撮影
2. 上記ツールでイラスト風に変換
3. 背景を削除（remove.bg など）
4. サイズを調整して保存

#### オプション4: イラストレーター依頼
**方法:**
- クラウドソーシング（ランサーズ、クラウドワークス等）
- 社内デザイナーに依頼
- 予算: 3,000円〜10,000円程度

---

## 🎨 デザインガイドライン

### ロゴ
- **カラー:** 会社の公式カラーを使用
- **背景:** 透過またはシンプルな背景
- **フォーマット:** PNG推奨、SVGも可

### アバター
- **表情:** 親しみやすい笑顔
- **服装:** ビジネススーツ（紺色推奨）
- **アクセサリー:** メガネ（知的な印象）
- **スタイル:** シンプル、フラットデザイン
- **年齢感:** 40代男性らしい落ち着き

---
