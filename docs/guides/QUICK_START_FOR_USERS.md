# ⚡ 5分でわかる！社内AI検索アプリの使い方

## 🎯 このアプリでできること

**社内の文書や情報を、AIに質問して簡単に探せます！**

---

## 🚀 3ステップで始める

### ステップ1: アクセス
```
https://internal-ai-search-app.streamlit.app
```
ブラウザで上記URLを開く

### ステップ2: ログイン
パスワードを入力
（パスワードは #ai-search-support で確認）

### ステップ3: 質問する
画面下の入力欄に質問を入力して送信！

---

## 💡 よく使う質問例

## 🎨 2つのモード

### 【モード1】社内文書検索
**使うとき**: 文書を探したい

**例**:
```
入力: 「有給休暇」
↓
結果: 関連ファイルのパス一覧
- data/規則規程について/就業規則.pdf
- data/社内ルール/休暇制度.docx
```

### 【モード2】社内問い合わせ（おすすめ！）
**使うとき**: 具体的に質問したい

**例**:
```
質問: 「有給休暇は年間何日取れますか？」
↓
回答: 「入社6ヶ月後から年間10日付与されます...」
情報源: data/規則規程について/就業規則.pdf
```

---

## ✨ 便利な機能

### 1. 続けて質問できる
```
質問1: 「有給休暇について教えて」
回答: 「年間10日付与されます...」

質問2: 「それは繰り越しできますか？」
回答: 「はい、翌年度に限り繰り越しできます」
```

### 2. 情報源が表示される
回答の下に、どの文書から情報を得たか表示されます。
必要に応じて元の文書を確認できます。

### 3. 履歴をクリアできる
新しい話題で質問したいとき：
左サイドバー → 「🔄 初期化」ボタン

---

## ⚠️ 3つの注意点

### 1. 回答は参考情報です
重要な決定の前は、必ず公式文書や担当部署に確認してください。

### 2. 社内専用です
URLやパスワードを外部に共有しないでください。

### 3. 利用制限があります
- 1日約750回の質問が可能（全社員で共有）
- 制限に達したら翌朝9時にリセット

## 🔧 よくあるトラブル

### 「情報が見つかりませんでした」と表示される
**対処法**:
1. キーワードを変えてみる
   - 「休み」→「有給休暇」
   - 「給料」→「給与」
2. モードを切り替えてみる
3. より一般的な言葉で質問

### 回答が返ってこない
**対処法**:
1. 30秒ほど待つ（処理中です）
2. ブラウザを更新（F5キー）
3. 質問を短くしてみる

### ログインできない
**対処法**:
1. パスワードを再確認
2. #ai-search-support で最新パスワードを確認
