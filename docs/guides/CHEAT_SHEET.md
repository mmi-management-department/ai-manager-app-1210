## 🆘 困ったときは

### よくあるトラブル

| 症状 | 対処法 |
|------|--------|
| 情報が見つからない | キーワードを変える |
| 回答が返ってこない | 30秒待つ → F5で更新 |
| ログインできない | パスワード再確認 |
| 制限に達した | 翌朝9時まで待つ |

### サポート窓口
**社内チャット**: `#ai-search-support`
**メール**: `ai-support@mm-international.co.jp`
**営業時間**: 平日 9:00〜18:00
