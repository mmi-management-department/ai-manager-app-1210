# 📖 社内AI検索アプリ - 利用ガイド

## 🎯 このアプリでできること

社内の文書や情報を、AIを使って簡単に検索・質問できます。

### できること
- ✅ 社内文書をキーワードで検索
- ✅ 自然な言葉で質問して回答を得る
- ✅ 情報源を明示（どの文書から回答したか）
- ✅ 会話履歴を保持（続けて質問できる）

---

## 🚀 アクセス方法

### ステップ1: アプリを開く

```
https://internal-ai-search-app.streamlit.app
```

### ステップ2: ログイン

パスワードを入力してログインします。
（パスワードは社内共有されています）

## 💡 使い方

### 【モード1】社内文書検索

関連する文書を探したいときに使います。

#### 使い方
1. 左サイドバーで「社内文書検索」を選択
2. 検索したいキーワードを入力
3. 関連するファイルのパスが表示されます

#### 例
```
入力: 「有給休暇」
結果: 
- data/規則規程について/就業規則.pdf (類似度: 0.85)
- data/既読規程以外の社内ルールについて/休暇制度.docx (類似度: 0.78)
```

#### こんなときに便利
- 「あの資料どこだっけ？」
- 「〇〇に関する文書を探したい」
- 「関連する資料を一覧で見たい」

---

### 【モード2】社内問い合わせ

具体的な質問をして回答を得たいときに使います。

#### 使い方
1. 左サイドバーで「社内問い合わせ」を選択
2. 自然な言葉で質問を入力
3. AIが文書を参照して回答します

#### 例
```
質問: 「有給休暇は年間何日取得できますか？」

回答: 
「当社の有給休暇は、入社6ヶ月後から年間10日付与されます。
その後、勤続年数に応じて最大20日まで増加します。」

情報源:
- data/規則規程について/就業規則.pdf
```

#### こんなときに便利
- 「〇〇について知りたい」
- 「△△の手順を教えて」
- 「××はどうなっていますか？」

## 📝 質問のコツ

### ✅ 良い質問の例

**具体的に聞く**
```
❌ 悪い例: 「休暇について」
✅ 良い例: 「有給休暇の申請方法を教えてください」
```

**背景を添える**
```
❌ 悪い例: 「手続きは？」
✅ 良い例: 「社員証を紛失した場合の再発行手続きを教えてください」
```

**1つずつ質問する**
```
❌ 悪い例: 「有給休暇と慶弔休暇と育児休暇について全部教えて」
✅ 良い例: 「有給休暇について教えてください」
           （回答後）「慶弔休暇についても教えてください」
```

### 質問の種類

| 質問タイプ | 例 |
|----------|---|
| 定義・説明 | 「〇〇とは何ですか？」 |
| 手順・方法 | 「△△の申請方法を教えてください」 |
| 期限・数値 | 「××の提出期限はいつですか？」 |
| 比較 | 「AとBの違いは何ですか？」 |
| 条件 | 「〇〇を利用できる条件は？」 |

## ⚠️ 注意事項

### 利用時の注意

1. **回答の確認**
   - AIの回答は参考情報です
   - 重要な決定の前は、必ず公式文書や担当部署に確認してください

2. **情報源の確認**
   - 回答には必ず情報源が表示されます
   - 元の文書を確認することをおすすめします

3. **機密情報の取り扱い**
   - 社内専用アプリです
   - URLやパスワードを外部に共有しないでください

4. **利用制限**
   - 1日約750回の質問が可能です
   - 制限に達した場合は翌朝9時にリセットされます

## 🆘 困ったときは

### よくあるトラブル

#### 1. アプリが開かない

**原因**: ネットワーク接続の問題

**解決策**:
- ブラウザを更新（F5キー）
- 別のブラウザで試す
- 社内ネットワークに接続されているか確認

#### 2. ログインできない

**原因**: パスワードが間違っている

**解決策**:
- パスワードを再確認（コピー＆ペースト推奨）
- 担当部署に問い合わせ

#### 3. 回答が返ってこない

**原因**: 処理中 or エラー

**解決策**:
- 30秒ほど待つ
- 「現在、回答を生成中です...」と表示される場合は処理中
- エラーが表示された場合は、質問を変えて再試行

#### 4. 「制限に達しました」と表示される

**原因**: 1日の利用制限に達した

**解決策**:
- 翌朝9時まで待つ
- 急ぎの場合は担当部署に直接問い合わせ

#### 5. 期待した回答が得られない

**原因**: 質問が曖昧 or 情報が不足

**解決策**:
- より具体的に質問する
- 背景情報を追加する
- モードを変更してみる（検索 ↔ 問い合わせ）

## 🎓 上級者向けTips

### Tip 1: 前の回答を参照する

```
質問1: 「有給休暇の付与日数を教えてください」
回答: 「入社6ヶ月後から年間10日...」

質問2: 「それは繰り越しできますか？」
→ AIは「有給休暇」の話だと理解して回答
```

### Tip 2: 範囲を絞る

```
❌ 「会社について教えて」
✅ 「会社の事業内容を3つ教えてください」
```

### Tip 3: 比較する

```
質問: 「特別休暇と有給休暇の違いを教えてください」
→ 違いを明確に説明してくれます
```

### Tip 4: 要約を依頼

```
質問: 「在宅勤務規程の要点を3つにまとめてください」
→ 長い文書を簡潔に理解できます
```
